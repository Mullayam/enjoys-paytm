PG KIT is an open source npm package that provides easiest payment gateway setup for paytm in your NodeJS/NextJs applications.

:::danger
Paytm is an Indian multinational financial technology company, that specializes in digital payments and financial services.
We created a simple package to integrate payment gateway in a easiest way to get rid of multiple line of code
:::

## Features

- Support NodeJs/NextJs -- (ReactJs Comming Soon)
- Modern, full-featured [client SDKs](/) for JavaScript, React, 
- Built for production

 

 
## Architecture
PG Kit is written in JavaScript and TypeScript.
